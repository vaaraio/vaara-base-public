# Security Policy

This repository is intentionally public-safe and excludes private execution logic, credentials, and internal infrastructure details.

## Reporting
If you find a security issue related to the contents of this repository, email: hello@vaara.io

## In scope
- Public page assets
- Non-sensitive frontend logic
- Public architecture notes included in this repository

## Out of scope
- Private pilot infrastructure
- Internal APIs and execution systems
- Deployment secrets and environment configuration
- Any environment or component not included in this repository
