# Quickstart: Publish Public GitHub Repo

Suggested repository name: `vaara-base-public`

## Git commands
From this directory, run:

```bash
git init
git add .
git commit -m "Initial public-safe Vaara Base surface"
git branch -M main
git remote add origin https://github.com/<your-username>/vaara-base-public.git
git push -u origin main
```

## Recommended GitHub settings
- Repo visibility: Public
- Description: `Public-safe Vaara Base product surface and frontend snapshot`
- Website: `https://base.vaara.io/demo`

## After push
- Use the repository URL in funding and ecosystem applications
- If needed, point reviewers to:
  - `README.md` for scope and positioning
  - `docs/ARCHITECTURE_PUBLIC.md` for architecture summary
  - live URLs for the product itself
